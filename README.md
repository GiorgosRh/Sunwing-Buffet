# Sunwing-Buffet
Buffet Menu and list of daily tasks

## BitChef Integration
Connected to [BitChef](https://github.com/GiorgosRh/BitChef) — the mobile app for kitchen staff at Helios & Ella.

The `kitchen.html` Task Manager syncs shift data in real time with the BitChef backend server (`helioskitch-mpdj7fvp.manus.space`).
